# 古诗词技能

为小朋友朗诵和讲解经典古诗词，培养文学素养和审美能力。

## 元数据

- **ID**: poetry
- **版本**: 1.0.0
- **图标**: 📜
- **关键词**: 古诗, 诗词, 唐诗, 宋词, 背诗, 朗诵
- **内容目录**: poems

## 触发条件

当用户表达以下意图时激活此技能：
- 背首古诗、念首诗、朗诵古诗
- 有什么古诗、古诗列表
- 唐诗、宋词、古诗词
- 静夜思、春晓等具体诗名

## 可用工具

### recite_poem

朗诵一首古诗词。

**参数：**
- `poem_name` (可选): 诗词名称或ID。如不指定则随机选择。

**使用示例：**
```
用户：背首古诗吧
调用：recite_poem()

用户：我想听静夜思
调用：recite_poem(poem_name="静夜思")
```

### list_poems

列出所有可用的古诗词。

**使用示例：**
```
用户：你会背什么诗？
调用：list_poems()
```

## 对话风格

使用此技能时的对话风格：
- 朗诵时声音抑扬顿挫、有韵律感
- 适当讲解诗词含义和背景
- 引导小朋友理解和欣赏
- 可以一起背诵互动

## 诗词资源

诗词存储在 `poems/` 目录下，每首诗一个 Markdown 文件。
