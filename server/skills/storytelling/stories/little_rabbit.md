# 小兔子乖乖

从前，有一只兔妈妈和三只小兔子住在森林边的小房子里。

一天，兔妈妈要出门采蘑菇。临走前，她对三只小兔子说："孩子们，妈妈出去采蘑菇，你们在家要乖乖的。记住，不管谁来敲门，都不要开门，知道吗？"

三只小兔子齐声说："知道了，妈妈！"

兔妈妈走后不久，一只大灰狼来到了小房子前。它捏着嗓子，学着兔妈妈的声音唱道：

"小兔子乖乖，把门儿开开，快点儿开开，妈妈要进来。"

小兔子们听了，从门缝里一看，发现外面是一条大尾巴，不是妈妈的短尾巴。于是，他们一起唱道：

"不开不开我不开，妈妈没回来，谁来也不开。"

大灰狼没办法，只好灰溜溜地走了。

过了一会儿，兔妈妈回来了。她唱道：

"小兔子乖乖，把门儿开开，快点儿开开，妈妈要进来。"

小兔子们从门缝里一看，看到了妈妈熟悉的短尾巴和长耳朵。他们高兴地打开门，扑进妈妈的怀里。

兔妈妈夸奖道："好孩子，你们真聪明！"

---

**故事启示：** 小朋友们要记住，不认识的人来敲门，一定不能随便开门哦！要学会保护自己。
