# 丑小鸭

在一个美丽的农场里，鸭妈妈孵出了一群小鸭子。可是，其中有一只小鸭子长得特别不一样——他的身体很大，羽毛是灰色的，看起来很丑。

其他小鸭子都嘲笑他："你长得真丑！"农场里的鸡、鹅也都不喜欢他。丑小鸭很伤心，他决定离开农场。

丑小鸭走啊走，来到一片沼泽地。野鸭们看见他，也说："你长得太丑了，离我们远点！"丑小鸭只好继续往前走。

秋天到了，树叶变黄了。丑小鸭看见一群美丽的白天鹅飞过天空，他们的羽毛洁白如雪，脖子又长又优雅。丑小鸭心想："要是我能变成那样该多好啊！"

寒冷的冬天来了，丑小鸭几乎被冻僵在冰面上。幸好一位农夫救了他，带他回家取暖。

春天终于来了。丑小鸭来到湖边，看见水中自己的倒影——他惊呆了！他不再是那只灰色的丑小鸭了，而是变成了一只美丽的白天鹅！

其他天鹅游过来，围着他说："欢迎你，美丽的新朋友！"

丑小鸭终于找到了属于自己的地方。他快乐地想："当初我做梦也没想到会有这么多的幸福！"

---

**故事启示：** 不要因为别人的嘲笑而看轻自己，每个人都有自己的闪光点，只要坚持努力，总有一天会绽放光彩。
