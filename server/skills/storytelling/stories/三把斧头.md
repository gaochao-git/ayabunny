# 三把斧头

从前，在一片美丽的森林边上住着一个特别诚实善良的小伙子，他有一把非常锋利的铁斧头。每天他都要带着这个铁斧头上山去砍柴，然后，再挑到山下去卖，卖了钱之后可以给家里买米买菜，他的日子就是这样一天天度过的。

有一天，这个小伙子呢，在去河边砍柴的时候，一不小心手一滑把自己的斧头掉进了河里。

就在这时候，河里住着神仙出现了，他了解了事情之后，觉得这小伙子特别的可怜，于是便沉到河里，一会儿他就上来了，捞起了一把用金子做的斧头，问小伙子说，“这一把金斧头是你的吗？”

小伙摇了摇头，“这不是我的斧头。”

接着，这个神仙又沉到河里，一会儿捞出一把银子做的斧头来，问道，“那这把银斧头是你的吗？”

这个小伙子看了，依旧摇摇头说，“不是！”

这个神仙呢，便第三次沉到河里去捞起了一把铁斧头，问，“那这一把是你的斧头吗？”

小伙子看了看，是自己的斧头，跳起来开心地说，“没错没错！这个才是我的斧头，太谢谢您了！”

神仙特别赞赏这个小伙子为人很诚实，于是就把那个金子做的斧头和银子做的斧头也作为礼物送给了他，要知道那可是非常非常值钱的呀！

小伙子就带着三把斧头回到了家里，同时他也把遇到的这个事儿告诉了乡亲们，可是有一个人特别的贪财，他也听说了这个事情，并特别的眼红，决定也去河边碰碰运气。

他跑到河边，故意把自己的铁斧头扔进了河里，然后就坐在河岸上假装哭了起来，“我的斧头啊，我以后怎么过日子啊？”
