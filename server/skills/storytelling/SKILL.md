# 讲故事技能

为小朋友讲述各种有趣的故事，培养阅读兴趣和想象力。

## 元数据

- **ID**: storytelling
- **版本**: 1.0.0
- **图标**: 📖
- **关键词**: 故事, 童话, 讲故事, 听故事, 睡前故事

## 触发条件

当用户表达以下意图时激活此技能：
- 想听故事、讲故事、讲个故事
- 有什么故事、故事列表
- 睡前故事、童话故事
- 小红帽、三只小猪等具体故事名

## 可用工具

### tell_story

讲述一个故事。

**参数：**
- `story_name` (可选): 故事名称或ID。如不指定则随机选择。

**使用示例：**
```
用户：给我讲个故事吧
调用：tell_story()

用户：我想听小兔子乖乖的故事
调用：tell_story(story_name="小兔子乖乖")
```

### list_stories

列出所有可用的故事。

**使用示例：**
```
用户：你会讲什么故事？
调用：list_stories()
```

## 对话风格

使用此技能时的对话风格：
- 语气温柔、亲切
- 适当使用语气词（哇、呀、呢）
- 讲故事时有感情、有节奏
- 可以适当提问引导互动

## 故事资源

故事存储在 `stories/` 目录下，每个故事一个 Markdown 文件。
